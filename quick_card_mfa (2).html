Ajout d’une authentification multifacteur

Dans le cas où l’utilisation d’un téléphone portable pour gérer l’authentification multifacteur n’est pas possible ou souhaitable, une solution alternative est d’utiliser votre navigateur.

Pour ce faire, installez l'extension suivante pour Google Chrome ou Microsoft Edge :

https://chromewebstore.google.com/detail/authenticator/bhghoamapcdpbohphigoooaddinpkbai

Nota : si cette extension est déjà installée, il est inutile de la réinstaller

Ajouter votre premier compte

Ce guide explique comment ajouter un compte à l’extension Authenticator.

Conditions préalables :

Installer l’extension Authenticator

Lorsque vous installez l’extension Authenticator pour la première fois dans Chrome ou Edge, il est préférable d’épingler son icône à la barre d'outils.

Attention : Utilisez le navigateur installé sur votre poste en priorité.

Ajouter un compte via un QR code

Cliquez sur l'icône l’extension Authenticator dans votre navigateur.

Cliquez sur l'icône de numérisation dans le coin supérieur droit de la fenêtre.

Cliquez et faites glisser pour dessiner un carré qui couvre complètement le QR code.

Une fois le nouveau compte ajouté, vous verrez une nouvelle entrée dans la fenêtre.

Le numéro à 6 chiffres est votre mot de passe à usage unique actuel, le cercle dans le coin inférieur droit vous indique le temps restant pour le mot de passe actuel. Si vous souhaitez ajouter ce compte dans une application mobile, vous pouvez cliquer sur l'icône QR code dans le coin supérieur droit de l'entrée pour afficher le QR code de ce compte.

Ajouter un compte manuellement

Cliquez sur l'icône de l’extension Authenticator dans votre navigateur.

Cliquez sur l'icône en forme de crayon en haut à droite de la fenêtre.

Cliquez sur le bouton + en haut à droite de la fenêtre. Sélectionnez Ajouter un compte

Saisissez le nom du site Web auquel ce compte est associé dans le champ Émetteur, le secret du compte qui vous a été donné dans le champ Secret, et enfin, cliquez sur Ok.

Une fois le nouveau compte ajouté, vous verrez une nouvelle entrée dans la fenêtre.

Copier un compte sur une autre session

Ce guide explique comment copier un compte de l’extension Authenticator vers une autre session.

Cliquez sur l'icône l’extension Authenticator dans votre navigateur.

Sur le code qui doit être copié sur une autre session, cliquez sur l'icône en forme de petit QR code.

Un QR code doit apparaitre en dessous du code à copier

Faites une capture d’écran, coller l’image dans un brouillon de votre messagerie et enregistrez-le sans l’envoyer.

Ouvrez votre messagerie sur le poste de destination

Ouvrez le brouillon

Répétez la procédure “Ajouter un compte via un QR code”
